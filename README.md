# Project Based Problem UAS DP 5 Kelompok 8 (Kalkulator Kalori)
Kalkulator kalori adalah kalkulator untuk menghitung kebutuhan kalori harian berdasarkan data fisik dan aktivitas seseorang.
Di kalkulator ini kami menggunakan rumus BMR, yaitu rumus yang paling umum dan sering digunakan untuk menghitung kalori.
